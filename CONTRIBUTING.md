# Contributing to Galleta-de-la-Fortuna_Random-Phrases-React 🍪

First of all, thank you for considering contributing to this project! It's people like you who make the open-source community such an amazing place to learn and create.

## 🚀 How Can I Contribute?

### Reporting Bugs 🐛
If you find any UI glitches or logical errors (e.g., the background doesn't change or phrases repeat incorrectly), please create an issue in the repository including:
* A clear and descriptive title.
* Describe the bug.
* Mention your browser and screen resolution (if it's a styling issue).

### Suggesting Enhancements 💡
Do you have an idea to optimize Galleta-de-la-Fortuna_Random-Phrases-React 🍪 or Want to add more fortunes?
* Create an **Issue** explaining the enhancement.
* If you plan to implement it, please wait for discussion before sending a Pull Request.

### Pull Requests 🏗️
1. Fork the repository.
2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
3. Since this uses **Vite**, make sure to run `npm run dev` to test your changes locally.
4. Commit your changes using **Conventional Commits** (e.g., `feat: add social login`).
5. Push to the branch (`git push origin feature/AmazingFeature`).

## ⚖️ Style Guide
* Use functional components and hooks (`useState`).
* Maintain the "LED" aesthetic and responsive design.
* Keep documentation (README) updated if you add new features.

# Contribuyendo a: Galleta-de-la-Fortuna_Random-Phrases-React 🍪

En primer lugar, ¡gracias por considerar contribuir a este proyecto! Son personas como tú las que hacen de la comunidad de código abierto un lugar increíble para aprender y crear.

---

## 🚀 ¿Cómo puedo contribuir?

### Reportar Errores (Bugs) 🐛
Si encuentras fallos en la interfaz o errores lógicos (ej: el fondo no cambia o las frases se repiten de forma incorrecta), por favor crea un "Issue" en el repositorio incluyendo:
* Un título claro y descriptivo.
* Una descripción del error.
* Tu navegador y resolución de pantalla (especialmente si es un problema de diseño).

### Sugerir Mejoras 💡
¿Tienes una idea para optimizar Galleta-de-la-Fortuna_Random-Phrases-React 🍪 o quieres añadir más frases de la fortuna?
* Crea un **Issue** explicando la mejora.
* Si planeas implementarla, por favor espera a que se discuta en el Issue antes de enviar un Pull Request.

### Pull Requests 🏗️
1. Haz un **Fork** del repositorio.
2. Crea tu rama de mejora (`git checkout -b feature/IncreibleMejora`).
3. Como este proyecto usa **Vite**, asegúrate de ejecutar `npm run dev` para probar tus cambios localmente.
4. Realiza tus commits usando **Conventional Commits** (ej: `feat: add social login`).
5. Haz **Push** a tu rama (`git push origin feature/IncreibleMejora`).

## ⚖️ Guía de Estilo
* Usa componentes funcionales y hooks (`useState`).
* Mantén la estética "LED" y el diseño responsivo.
* Mantén la documentación (README) actualizada si añades nuevas funcionalidades.
